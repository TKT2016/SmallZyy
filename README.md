# SmallZyy
仿照微软Small Basic开发的Z语言中文编程工具，实现了微软Small Basic 的大部分功能
-------------------
##使用方法

1 按照.NET Framework最低4.5版本
2 下载SmallZyy.zip
3 解压SmallZyy.zip
4 运行SmallZyy.exe
